# Forge Framework v2

 A lightweight dependency injection container inspired by ServiceBag.

WARNING:
- Not battle tested
- Never used in production

NOTE FROM DEV:
It's not a perfect module, it was inspired by service bag, but I just really didn't want the bloat from it.
I only wanted the ServiceBag itself but it was also tied to the framework.
I am still yet to use this in actual game, I will be iterating it rapidly, things will break, but I believe trial by fire is #1.
This module was made after v1 frustrated me with having to define types for every single new service, method and metadata.
